How to Use This Application

  1. Add Nodes: Click the "Add Node" button to add a new node to your flowchart. Each node represents a step or process in your logistics workflow.
  2. Position Nodes: Drag and drop nodes to position them on the flowchart canvas. Arrange them in a logical sequence.
  3. Label Nodes: Double-click on any node to edit its label. Enter descriptive text that explains the process step.
  4. Connect Nodes: Click on a node and drag to another node to create a connection between them, showing the flow of your logistics process.
  5. Save Your Work: Use the "Save PNG" button to save your flowchart as an image file, or "Save PDF" for a document format that's easy to share.
